# Changelog

All notable changes to this project will be documented in this file.

The format is based on Keep a Changelog (https://keepachangelog.com/en/1.1.0/),

## [0.0.1] - 2025-11-18

### Added
- Initial public release of the DNS MCP server.
- Node/TypeScript implementation with CLI entry `dns-mcp-server`.
- Knowledge modules: BIND, NSD, Unbound, DNSSEC, DANE/TLSA, registrar operations, troubleshooting.
- Tools: config generator & validator, dig wrapper, DNS debugger, zone analyzer.
- Documentation: README and QUICK_START.
